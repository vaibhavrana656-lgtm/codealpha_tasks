# Data Sources

## Books
Books to Scrape is a public sandbox designed for scraping practice. The live scraper is included in Task 1, while the bundled CSV makes the project viewable offline.

## Sentiment
Kotzias, D. (2015). Sentiment Labelled Sentences. UCI Machine Learning Repository. DOI: 10.24432/C57604.

The Amazon subset has 1,000 labelled reviews: 500 positive and 500 negative.
